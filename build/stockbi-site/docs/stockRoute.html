Endpoints de Estoque

Rotas da entidade Stock

MétodoRotaDescrição
GET /stock/find-allRetorna todos os registros de estoque.
GET /stock/find-by/id=:idRetorna um registro de estoque específico com base no ID fornecido.
GET /stock/find-by/establishment=:establishment_idRetorna todos os registros de estoque de um estabelecimento específico.
POST /stock/create/establishment=:establishment_idCria um novo registro de estoque associado a um estabelecimento específico.
PATCH /stock/update/id=:id/establishment=:establishment_idAtualiza um registro de estoque específico com base no ID e estabelecimento fornecidos.
DELETE /stock/delete/id=:idExclui um registro de estoque específico com base no ID fornecido.

Exemplo de solicitação de criação de registro de estoque:

POST /stock/create/establishment=12345
Content-Type: application/json

{
    "code": "03-OLEOS"
}

Exemplo de resposta da criação de registro de estoque:

HTTP/1.1 201 Created
Content-Type: application/json

{
    "newStock": {
        "identifiers": [
            {
                "id": 4
            }
        ],
        "generatedMaps": [
            {
                "id": 4
            }
        ],
        "raw": [
            {
                "id": 4
            }
        ]
    },
    "success": true
}

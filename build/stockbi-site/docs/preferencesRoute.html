Preferences

Endpoint de Preferencias

Rotas da entidade Preferences

MétodoRotaDescrição
POST /preferences/create/establishment=:establishment_idCria uma nova preferência associada a um estabelecimento específico.
GET /preferences/find-one/:idRetorna uma preferência específica com base no ID fornecido.
GET /preferences/find-allRetorna todas as preferências.
PATCH /preferences/:idAtualiza uma preferência específica com base no ID fornecido.
DELETE /preferences/:idExclui uma preferência específica com base no ID fornecido.
GET /preferences/find-one-by-userRetorna uma preferência com base no usuário atual.

Exemplo de solicitação de criação de preferência:

POST /preferences/create/establishment=12345
Content-Type: application/json

{
    "theme": "light",
    "language": "português"
}

Exemplo de resposta da criação de preferência:

HTTP/1.1 201 Created
Content-Type: application/json

{
    "newPreference": {
        "theme": "light",
        "language": "português",
        "establishment": {
            "id": 16,
            "name": "Filial 032321",
            "code": "STKBI_F01"
        },
        "id": 2
    },
    "success": true
}
